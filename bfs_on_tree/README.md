Find the target value using Breadth First Search in a tree.

###### Example

```
Given the root node of the following tree as input

       1
     /   \
    2     3
   / \     \
  4   5     6
 / \ / \   / \
3  8 9 10 11 12

And the target of 3,

your code should search: 1, 2, 3 and then return True
```
